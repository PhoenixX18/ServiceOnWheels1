# 🚗 Service on Wheels

A full-stack roadside assistance web application built using Angular, Spring Boot, MongoDB, and JWT authentication.

## Features

- User Registration & Login
- JWT Authentication
- Secure Password Hashing (BCrypt)
- Role-based Route Protection
- Service Request Creation
- Request History Tracking
- MongoDB Database Integration
- Responsive Angular Frontend
- REST API Backend

## Tech Stack

Frontend:
- Angular
- TypeScript
- CSS
- Angular Guards
- HTTP Interceptors

Backend:
- Java
- Spring Boot
- Spring Security
- JWT
- BCrypt
- Maven

Database:
- MongoDB

Tools:
- Git
- GitHub
- Postman

## Run Locally

Backend:

```bash
mvn spring-boot:run
```

Frontend:

```bash
npm install
ng serve
```

Frontend:
http://localhost:4200

Backend:
http://localhost:8081
